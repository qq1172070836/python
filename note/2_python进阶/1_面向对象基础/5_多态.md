## 面向对象三大特性——多态
### 多态的概念
    多个对象公用一个接口
    一个接口多种形态

### 1_函数实现多态
    ```
    class Dog(object):
        def sound(self):
            print('汪汪汪。。。')

    class Cat(object):
        def sound(self):
            print('喵喵喵。。。')

    def make_sound(animal_ojb):
        # 统一调用的接口
        animal_ojb.sound()

    d = Dog()
    c = Cat()
    make_sound(d)
    make_sound(c)
    ```

### 2_抽象类实现多态

    ```
    class Document(object):
        def __init__(self, name):
            self.name = name

        def show(self):
            raise NotImplementedError('必须重新父类方法')

    class Pdf(Document):
        def show(self):
            return 'show pdf contents'

    class Word(Document):
        def show(self):
            return 'show word contents'

    pdf_obj = Pdf('财务账单.pdf')
    word_obj = Word('简历.word')
    print(pdf_obj.show())
    print(word_obj.show())
    ```
