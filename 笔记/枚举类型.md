####枚举类

    from enum import Enum
    class VIP(Enum):
        YELLOW = 1
        GREEN = 2
        BLACK = 3
        RED = 4
        
####枚举类型
    
    print(type(VIP.YELLOW))
    =>
    enum类型
    
    枚举名称
    VIP.YELLOW.name
    
    枚举值
    VIP.YELLOW.value
    
    VIP(1)
    =>
    VIP.YELLOW
    
    枚举类型，标签不能重复，值可以重复
    
    遍历枚举类型
    for i in VIP:
        print(i)
